## WithdewHua's Dotfiles

## Intro

- Personal dotfiles for macOS/Linux;
- Managed by [chezmoi](https://github.com/twpayne/chezmoi)

## How to use

### Installation

```bash
bash <(curl -fsLS git.io/chezmoi) -b /usr/local/bin -- init --apply --verbose WithdewHua
```

### Update

```bash
chezmoi update
```
