## WithdewHua's Dotfiles

## Intro

- Personal dotfiles for macOS/Linux;
- Managed by [chezmoi](https://github.com/twpayne/chezmoi)

## Usage

### Installation

```bash
bash <(curl -fsLS chezmoi.io/get) -b /usr/local/bin -- init --apply --verbose WithdewHua
```

### Update

```bash
chezmoi update
```
